# Workorders React App

## Installing
Needs internet connection
```
git clone https://github.com/pxai/react.workorders.git
cd react.workorders
npm install
```

## Starting
Open the console and:
```
npm run start
```

## Testing
Just type:
```
npm run test
```

## Production
If you want it ready for prodction:
```
npm run build
```
